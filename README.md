
# crowdsourceLandingPage

landing page for crowdsource project

## how

### install

```
git clone https://github.com/rosie-brigham/crowdsourceLandingPage
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
